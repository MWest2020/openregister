\OCA\OpenRegister\Service UploadService

Service for handling file and JSON uploads.

This service processes uploaded JSON data, either directly via a POST body, from a provided URL, or from an uploaded file. It supports multiple data formats (e.g., JSON, YAML) and integrates with schemas and registers for database updates.
