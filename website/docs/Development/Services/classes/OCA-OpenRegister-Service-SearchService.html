recursiveRequestQueryKey()

recursiveRequestQueryKey(array  $vars, string  $name, string  $nameKey, string  $value) : void

This function adds a single query param to the given $vars array. ?$name=$value Will check if request query $name has [...] inside the parameter, like this: ?queryParam[$nameKey]=$value.

Works recursive, so in case we have ?queryParam[$nameKey][$anotherNameKey][etc][etc]=$value. Also checks for queryParams ending on [] like: ?queryParam[$nameKey][] (or just ?queryParam[]), if this is the case this function will add given value to an array of [queryParam][$nameKey][] = $value or [queryParam][] = $value. If none of the above this function will just add [queryParam] = $value to $vars.

Parameters

array $vars

The vars array we are going to store the query parameter in
string $name

The full $name of the query param, like this: ?$name=$value
string $nameKey

The full $name of the query param, unless it contains [] like: ?queryParam[$nameKey]=$value
string $value

The full $value of the query param, like this: ?$name=$value

Returns

void —
